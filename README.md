# S-P-500-Momentum-Enhanced-Indexing-Strategy

一. 执行摘要

本报告旨在展示一套基于“风险调整动量”的量化选股策略。策略通过剔除近期波动率影响，提取中长期价格惯性，并结合缓冲规则（Buffer Rule）以降低组合周转率。

二. 投资哲学与研究动机

动量效应（Momentum Effect）在学术界与实务界均已被广泛验证。本策略的核心逻辑在于：

1、风险溢价识别：单纯的价格上涨可能伴随高波动。本策略通过波动率归一化处理，剥离因波动率带来的虚假动量，捕捉真实的超额收益。

2、路径依赖控制：引入剔除最近1个月效应的机制，避开价格短期反转风险，确保动量特征的持续性。
