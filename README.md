# S-P-500-Momentum-Enhanced-Indexing-Strategy

一、执行摘要

本报告旨在展示一套基于“风险调整动量”的量化选股策略。策略通过剔除近期波动率影响，提取中长期价格惯性，并结合缓冲规则（Buffer Rule）以降低组合周转率。



二、投资哲学与研究动机

动量效应（Momentum Effect）在学术界与实务界均已被广泛验证。本策略的核心逻辑在于：

1、风险溢价识别：单纯的价格上涨可能伴随高波动。本策略通过波动率归一化处理，剥离因波动率带来的虚假动量，捕捉真实的超额收益。

2、路径依赖控制：引入剔除最近1个月效应的机制，避开价格短期反转风险，确保动量特征的持续性。

5、组合优化与执行

1、动态选股与缓冲机制：

为解决频繁调仓带来的交易摩擦，应用 20% 换手缓冲规则。若成分股得分排名在一定区间内波动，将保留现有持仓，以降低年度换手率。

2、权重分配：

采用“市值 × 动量得分”加权，并在个股层面实施 9% 的权重上限限制，有效分散单一标的带来的非系统性风险。



四、风险与局限性

1、市场环境风险：在市场风格发生剧烈切换（如大幅V型反转）时，动量策略可能面临阶段性回撤。

2、交易成本假设：本回测未显式扣除双边印花税及佣金，实际运行中，受缓冲规则影响的调仓成本应纳入进一步评估。

3、历史回溯局限：模型基于历史数据训练，未来市场结构变化可能导致因子失效。
